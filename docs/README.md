# このレポジトリは初めてスマホを使う人向けに作った、説明用HPを書いたものです。

## このHPについての概要
- このHPは、初めてスマホを使う方向けに、スマホの使い方を記したHPです。
- 手書きの文字やスマホを直撮りした動画があり、見にくいところがございますが、"味"だと思ってください。
- スマホで下のQRコードを読むことで、そのHPにアクセスすることができますので、ぜひご覧ください。
- スマホでQRコードを読む方法が分からない方は、QRコードの下にある説明書を読んでください。

## HPにアクセスするQR code
![QR_code](../picture/qr.png)
https://xingping0721.github.io/pixel_setup.github.io/

## QR codeの読み方
- 電源ボタンを２回連続で短く押してください。
- カメラが開きます。
- QRコードにカメラを向けると、自動的に読み込まれます。
- 画面上に表示されたリンクをタップしてHPにアクセスしてください。

- 電源ボタンは下の写真で分かるように、スマホ右側面の一番上のボタンです。
![hardware_1](../picture/hardware_1.jpg)

## 使い方のコツ
1. **まずはホーム画面から始めましょう** - スマホを初めて使う方は、まずホーム画面の説明を見てください。
2. **一つずつ学んでいきましょう** - 全部をすぐに覚える必要はありません。必要な機能から使ってみましょう。
3. **わからないことは何度でも見てください** - いつでもこのHPに戻って確認できます。
4. **家族や友人に聞いてもいいです** - スマホの使い方について、周りの人に相談することも大切です。

## ご注意
- このHPの内容は、Google Pixel シリーズのスマホを想定して作られています。
- 他のメーカーのスマホの場合、操作方法が異なる場合があります。
- スマホの操作に不安があれば、認定の販売店や家族に相談してください。
